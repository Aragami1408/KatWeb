It works!

This is the example page used to test the correct operation of the KatWeb HTTP Server server after installation.

KatWeb HTTP Server is a web server designed for the modern web, created by kittyhacker101.

Why am I here?

If you are a normal user of this web site and don't know what this page is about, this probably means that the site is currently unavailable due to maintenance.
If the problem persists, please contact the site's administrator.

If you are the administrator of this site, continue reading below.

What do I do next?

You should replace this this file and install a HTTPS certificate before continuing to operate your HTTP server.
The locations of where these files should be placed are shown below. Note that all paths are relative to the install path.

/html/ - Document root of server.
/ssl/ - Server HTTPS certificates.
/error/ - Server error pages.
/conf.json - Server config file.
